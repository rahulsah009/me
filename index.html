RydeOn - Ride Booking Web Application

RydeOn is a web-based ride-booking application similar to Uber and Ola, developed using the MERN stack. It allows users to book rides in real-time through a map-based interface, track drivers live, make secure payments, and verify bookings via OTP authentication. With a user-friendly design and seamless ride management system, RydeOn offers a smooth and efficient experience for both riders and drivers.

React.js, Tailwind Css, Node.js, Express.js, MongoDB, Rest APIs, JWT authentication
